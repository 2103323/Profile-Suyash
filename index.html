FeedBack Analysis

This project focuses on building a multi-label classification model using BERT to categorize product descriptions into multiple relevant categories. The dataset includes item descriptions with associated Level 1 Factors (labels). After preprocessing the text and encoding the labels, a fine-tuned BERT model (`bert-base-uncased`) was used for training. The model applies a sigmoid activation function to handle the multi-label nature of the task and uses binary cross-entropy loss. Predictions are made on unseen test data, and the results are saved for evaluation.
